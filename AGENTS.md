# Agent Instructions

This repository contains a React/Vite frontend and an Express backend.

## Project Rules

- Keep generated dependencies, build output, logs, and local `.env` files out of
  version control.
- Keep public documentation current when user-facing behavior changes.
- Run linting, tests, coverage, dependency audit, and secret scanning before
  merging.
- Preserve the AGPLv3 license.

## Useful Commands

```powershell
npm run dev
npm run lint
npm run test
npm run coverage
npm run audit
npm run secrets
```
