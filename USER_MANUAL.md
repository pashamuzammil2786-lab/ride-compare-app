# User Manual

FareFinder helps users compare ride estimates and shopping results before
opening the original provider platform.

## Start The App

```powershell
npm run dev
```

Open the frontend at `http://localhost:5173`.

## Create An Account

1. Select **Signup**.
2. Enter your name, email address, and password.
3. Sign in to access the comparison modules.

## Compare Rides

1. Choose **Ride Compare**.
2. Enter pickup and destination locations.
3. Select a ride type: All, Bike, Auto, or Car.
4. Select **Compare rides**.
5. Open the preferred provider from the result card.

## Compare Products

1. Choose **E-commerce**.
2. Enter product keywords or upload a product photo.
3. Select a size when needed.
4. Select **Compare products**.
5. Open the marketplace result you prefer.

## Notes

Ride prices are estimates for demo use. Production pricing requires official
provider APIs or approved partner integrations.
